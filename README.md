I was able to practice some more of Flexbox properties, styling and using CSS variables to set default colors.

    --primary-soft: hsl(215, 51%, 70%);

    --neutral-medium: hsl(216, 50%, 16%);
    --neutral-soft: hsl(215, 32%, 27%);
